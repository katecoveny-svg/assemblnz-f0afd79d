# Project Memory

## Core
Draft-Only mode: all consequential outputs (social, payments, etc.) require explicit human sign-off.
Strict Privacy Act 2020 & Te Mana Raraunga compliance. Mask PII; use masked-input forms for secrets.
No generic AI terms ('AI', 'brain', 'streamline', etc.). Use NZ English (e.g., 'analyse').
8 ketes (LOCKED): Manaaki, Waihanga, Auaha, Arataki, Pikau, Hoko, Ako (industry) + Tōro (consumer/family). No 'Toro' macron drop — use 'Tōro'. Retired: Hanga, Pakihi, Waka, Hangarau, Hauora, Te Kāhui Reo, Whenua, Ora, Kāinga.
Pricing (LOCKED 2026-04-20, NZD ex GST): Tōro $29/mo · Operator $1,490+$590 (1 kete) · Leader $1,990+$1,290 (2 ketes) · Enterprise $2,990+$2,890 (7+Tōro) · Outcome from $5,000. Cross-cutting on every paid tier: AROHA·SIGNAL·SENTINEL. NO multi-kete % discount — only ANNUAL12 (12% off annual prepay). Operator-as-platform = same Operator SKU at /platform (not /business).
Light Glass System: white-ice-blue bg (linear #FFFFFF→#F4FAFC→#E8F4F6), glass rgba(255,255,255,0.65), soft teal #4AA5A8, text #1F4E54. No dark mode.
Status colors: Error (#C85A54), Success (#4AA5A8), Pending (#5A7A9C info-blue, NOT amber), Info (#5A7A9C). NO ochre/yellow anywhere.
No generic emojis (🟢🟡🔴🧠✨⚡✅❌ etc.) — always use Lucide icons coloured with the status palette.
Do NOT use actual corporate brand names (HubSpot, Xero); use generic replacements.
5-stage Governance Pipeline applied to AI interactions (Kahu, Iho, Tā, Mahara, Mana).
Mobile-first 'phone-screen' chat UI with bottom directional rounded-2xl message bubbles.
Construction kete pack ID is "waihanga" (not "hanga"). Guard script at scripts/check-kete-names.ts.
ARATAKI is the Automotive kete — NOT privacy/business compliance.
Headings Inter 300-400 (never bold), body Inter 400-500. Text #1A1D29 never pure black.
Pearl homepage (/) palette LOCKED: Warm Pearl #FAF6EF canvas · Linen #F4EFE6 alt · Forest Ink #0F2A26 text (deep pounamu, never black) · Pounamu #1F4D47 accent · Candle-warm #F8E9C4 sparkles. Hero kete must sit nested inside HeroCloud, not floating alone.
FORBIDDEN CLAIMS (Q2 2026): no "trained on 50+ Acts", no "enterprise-grade", no "world-class/best-in-class", no "unlock" verb, no "fully sovereign", no te reo as H1, no emoji in marketing. Partners (Kererū/Vocus/Spark) → "exploring relationships with NZ-owned infrastructure providers", never named as confirmed.
Positioning narrative: "Intelligent tools for every NZ operator — accessibility-first." Three pillars: (1) made accessible $29→Outcome, (2) NZ-specific by default, (3) evidence packs not chat. Tagline: "Intelligence, shaped for Aotearoa — at a price every operator can afford."
Canonical kete visual is locked: fluffy white FeatherKete with subtle coloured rim washes only; never generate alternate kete imagery unless explicitly requested.
