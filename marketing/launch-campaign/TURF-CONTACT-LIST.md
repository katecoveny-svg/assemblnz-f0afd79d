## BULK EMAIL SETUP — Brevo (free tier)

### Steps:
1. Go to brevo.com and create free account (300 emails/day)
2. Import the Tier 2 + Tier 3 contacts as a CSV
3. Create an email template using the "Regional Sports Trust" or "Individual Club" template from TURF-OUTREACH-EMAILS.md
4. Use merge tags for personalisation: {{organisation_name}}, {{region}}
5. Send in batches of 50-100 per day to stay within free tier
6. Track open rates and click-through to assembl.co.nz/chat/sports

### CSV Format for Import:
```
email,first_name,organisation,region,type
info@sportwaikato.org.nz,Team,Sport Waikato,Waikato,regional_trust
info@aucklandrugby.co.nz,Team,Auckland Rugby,Auckland,provincial_union
```
