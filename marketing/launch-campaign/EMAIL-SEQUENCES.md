## Email Design Notes

### Brand Guidelines for Email

| Element | Specification |
|---------|--------------|
| From name | Kate from Assembl |
| From email | kate@assembl.nz |
| Reply-to | kate@assembl.nz (monitored) |
| Font | System font stack (clean, fast-loading) |
| Primary colour | #6C2BD9 (Assembl purple) |
| Button colour | #6C2BD9 with white text |
| Button style | Rounded corners, large padding, mobile-friendly |
| Layout | Single column, max 600px wide |
| Images | Minimal -- one header image max per email |
| Footer | Assembl, Auckland, New Zealand | Unsubscribe | Privacy Policy |

### Personalisation Variables

| Variable | Source | Fallback |
|----------|--------|----------|
| [First Name] | Signup form | "there" |
| [Agent Name] | First agent selected | "your agents" |
| [Plan Name] | Selected plan | "your plan" |
