## INTELLIGENCE LAYER

Connecting all 43 agents is a shared intelligence infrastructure:

```
+=============================================================================+
|                       ASSEMBL INTELLIGENCE LAYER                            |
+=============================================================================+
|                                                                             |
|  1. SHARED CONTEXT BUS (agent_context table)                                |
|     - Every agent reads/writes to a shared context store                    |
|     - When LEDGER flags a cashflow issue, FLUX knows to chase invoices      |
|     - When AROHA onboards a new hire, SIGNAL sets up IT access              |
|     - Cross-agent awareness without manual coordination                     |
|                                                                             |
|  2. PROACTIVE ALERTS (agent-intelligence edge function)                     |
|     - Agents monitor their domains and push alerts proactively              |
|     - ANCHOR alerts when legislation changes affect the business            |
|     - TERRA alerts when biosecurity risks emerge                            |
|     - LEDGER alerts before tax deadlines                                    |
|                                                                             |
|  3. DAILY BUSINESS BRIEFING                                                 |
|     - ECHO compiles a morning briefing from all agent signals               |
|     - Revenue snapshot (LEDGER + FLUX)                                      |
|     - Legal/compliance flags (ANCHOR + industry agents)                     |
|     - Team updates (AROHA)                                                  |
|     - Content performance (PRISM + ECHO)                                    |
|                                                                             |
|  4. ASSEMBL BUSINESS SCORE                                                  |
|     - Composite health score drawn from all agent data                      |
|     - Financial health (LEDGER, VAULT, MINT)                                |
|     - Legal compliance (ANCHOR, industry agents)                            |
|     - People health (AROHA, THRIVE)                                         |
|     - Growth trajectory (FLUX, PRISM)                                       |
|     - Security posture (SIGNAL)                                             |
|                                                                             |
|  5. BRAND DNA (shared brand profile)                                        |
|     - Unified tone, voice, and brand guidelines                             |
|     - Every agent speaks with consistent brand personality                  |
|     - Shared visual identity rules for all generated content                |
|     - Customer persona library accessible to all agents                     |
|                                                                             |
+=============================================================================+
```
